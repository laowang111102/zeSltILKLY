# Python学生就业管理系统 python1055

## 项目概述

本项目是一款基于Python的学生就业管理系统，集成了学院、专业、企业、岗位等多维度的信息管理功能，旨在为学校和学生提供一个便捷、高效的就业信息平台。

## 技术栈

- 前端：HTML、CSS、JavaScript、jQuery
- 后端：Python、Django
- 数据库：MySQL
- 开发工具：PyCharm、Navicat

## 运行环境

- Python 3.0以上
- MySQL 8.0

## 功能模块

- 学院信息管理
- 专业信息管理
- 企业信息管理
- 岗位信息管理
- 管理员信息管理
- 学生信息管理
- 学生教育经历
- 学生项目经历
- 学生投递记录
- 修改个人信息与密码

## 系统角色

- 管理员
- 教师
- 学生

## 目录结构

```
.
├── app             # 应用代码目录
│   ├── models.py   # 数据模型
│   ├── views.py    # 视图处理逻辑
│   ├── admin.py    # 管理后台配置
│   └── ...         # 其他应用相关代码
├── config          # 配置文件目录
├── db              # 数据库相关
├── static          # 静态资源目录
├── templates       # 模板文件目录
└── manage.py       # 管理脚本
```

## 部署步骤

1. 环境准备：确保Python 3.0以上和MySQL 8.0已安装。
2. 数据库配置：使用Navicat等工具创建数据库，并根据项目需求导入数据。
3. 项目配置：根据实际环境修改配置文件，如数据库连接信息等。
4. 运行项目：使用PyCharm或命令行运行`manage.py`脚本，启动项目。
